# Personal Site Starter

A minimal, accessible static site built with plain HTML & CSS. Ideal for a quick personal site: bio + links, portfolio/gallery, and a simple blog landing page.

## Structure

```
personal-site-starter/
├── index.html
├── about.html
├── portfolio.html
├── blog/
│   └── index.html
├── assets/
│   └── images/   # put your images here
└── styles.css
```

## Local preview

1. Open the folder in VS Code.
2. Install the **Live Server** extension.
3. Right‑click `index.html` → **Open with Live Server**.

## Deploy (GitHub Pages)

1. Create a new repo on GitHub and push these files.
2. In your repo: **Settings → Pages**. Set **Branch** to `main` and **/ (root)`**. Save.
3. Wait a minute; your site will be at `https://<username>.github.io/<repo>/`.

## Tips

- Optimize images before uploading (try Squoosh).
- Keep alt text meaningful.
- Use system fonts (fast!) or add a web font later.
